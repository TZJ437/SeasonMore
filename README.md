# 季节的旋律

一个离线可运行的二十四节气科普作品。项目用四季、节气原理、图集、典籍故事、干支、日期查询和互动测试串成完整展陈路径，适合课堂展示、课程作业或本地科普演示。

## 功能概览

- 首页：四季沉浸场景、春夏秋冬交互彩蛋、今日近节气与干支提示。
- 四季展厅：按春、夏、秋、冬组织全年风物、物候和图像内容。
- 节气原理：解释太阳黄经、公历日期稳定性、节气与农历的区别，以及地域物候差异和常见误区。
- 节气图集：二十四节气卡片和离线图片，可按季节筛选并进入详情页。
- 典籍故事：用“原文摘句 + 白话解释”的方式连接《月令》、天文、物候、农事和礼俗。
- 天干地支：展示十天干、十二地支、六十甲子和地支月份。
- 节令日历：查询指定日期的季节、附近节气、干支纪年与干支日。
- 互动测试：包含问答、拼图、诗句补全、配对、排序和翻牌小游戏。

## 运行方式

项目是纯静态页面，直接打开 `index.html` 即可浏览。

也可以在项目目录启动本地服务：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/index.html
```

## 说明

- 项目图片已本地化到 `images/photos`，断网也能展示主要图集内容。
- 节气日期、干支日等计算用于科普理解，采用常见近似规则，不替代精确天文历算。
- 默认“今天”会读取本机时间；如果本地系统时间被改到异常年份范围外，会回落到项目内置演示日期，避免页面被错误时间破坏。用户手动输入日期和年份仍可正常查询。
